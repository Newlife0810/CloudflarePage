快速開始

  1. 將此專案與 Git 儲存庫連結到 Cloudflare Pages。
  2. 確認 build command 與 output directory（例如：npm run build / public）。
  3. 在 Pages 控制台啟用自動部署，並設定自訂網域或預設子域名。
Branch: main

建議的下一步

  • 新增自訂域名並開啟 Cloudflare 的 DNS 管理。
  • 啟用 Pages Functions 或 Edge Functions 提升動態功能。
  • 設定環境變數（API_KEY 等）於 Pages 設定中。
